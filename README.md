
  # HR Management System Design

  This is a code bundle for HR Management System Design. The original project is available at https://www.figma.com/design/7p5NaqgVsG9P2PjnPVDPPd/HR-Management-System-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to Vercel

  This project is configured for Vercel with:
  - `buildCommand`: `npm run build`
  - `outputDirectory`: `dist`
  - SPA fallback routing for React Router (`/dashboard`, `/profile`, etc.)

  ### Steps

  1. Push this repo to GitHub/GitLab/Bitbucket.
  2. Import the project in Vercel.
  3. Keep the detected framework as `Vite`.
  4. Deploy.

  No environment variables are required for the current mock-data setup.
